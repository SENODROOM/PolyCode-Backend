## 📖 Topics Covered

### 1. I/O Statements
- `print()` — basic, sep, end, f-strings, .format(), % operator
- `input()` — string, int, float, multiple values
- Type conversion — int(), float(), str()
- File I/O — read, write, append

### 2. Conditional Statements
- `if` — single condition
- `if-else` — two paths
- `if-elif-else` — multiple conditions
- Nested `if` — conditions inside conditions
- Ternary operator — one-line if-else
- `match-case` — pattern matching (Java 3.10+)
- Comparison operators: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical operators: `and`, `or`, `not`

### 3. Repetitive Statements (Loops)
- `for` loop — range(), lists, strings, dicts
- `while` loop — condition-based repetition
- `break` — exit loop early
- `continue` — skip current iteration
- `pass` — placeholder
- `else` with loops
- Nested loops
- List comprehension

### 4. Functions
- Basic function definition and calling
- Parameters and arguments
- Return values (single and multiple)
- Positional arguments
- Keyword arguments
- Default arguments
- `*args` — variable positional arguments
- `**kwargs` — variable keyword arguments
- Lambda functions
- Recursive functions
- Scope — local vs global variables
- `global` keyword
- Docstrings
- Higher-order functions (map, filter, reduce)
