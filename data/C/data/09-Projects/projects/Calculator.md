# Calculator Project

A basic calculator that performs addition, subtraction, multiplication, and division operations.

## Features:

- Basic arithmetic operations (+, -, *, /)
- Input validation
- Error handling for division by zero
- Interactive menu-driven interface
- History of calculations
- Exit option

## How to Use:

1. Run the program
2. Choose an operation from the menu
3. Enter two numbers
4. View the result
5. Continue with more calculations or exit

## Learning Concepts:

- Basic input/output (scanf, printf)
- Control flow (switch, while)
- Functions for modularity
- Error handling
- Data validation

## Improvements You Can Make:

- Add more operations (modulus, square root, power)
- Implement expression parsing
- Add calculation history display
- Save calculations to a file
- Support decimal numbers with validation

## Compilation:

```bash
gcc -o calculator Calculator.c
./calculator
```

See Calculator.c for the implementation.
