# File I/O Example

## File Overview
Comprehensive demonstration of Java file I/O including basic operations, NIO.2 features, directory handling, serialization, properties files, and performance comparisons.

## Key Concepts

### Traditional I/O
- **File**: Legacy file handling
- **FileReader/Writer**: Character-based I/O
- **BufferedReader/Writer**: Buffered operations
- **FileInputStream/OutputStream**: Byte-based I/O

### NIO.2 Features
- **Path**: Modern path handling
- **Files**: Utility methods
- **DirectoryStream**: Directory iteration
- **File Attributes**: Metadata access

### File Operations
- **Create**: New files and directories
- **Read**: Various reading methods
- **Write**: Different writing approaches
- **Delete**: File and directory removal
- **Copy**: File duplication

### Advanced Features
- **Serialization**: Object persistence
- **Properties**: Configuration files
- **Directory Traversal**: Recursive operations
- **File Attributes**: Metadata handling

### Best Practices
- **Try-with-resources**: Automatic cleanup
- **Buffered operations**: Performance optimization
- **Exception handling**: Robust error management
- **Path normalization**: Cross-platform compatibility
