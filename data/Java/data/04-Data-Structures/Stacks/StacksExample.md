# Stacks Example

## File Overview
Demonstrates stack implementations (built-in, custom linked list, array-based) and real-world applications like palindrome checking, bracket balancing, and expression evaluation.

## Key Concepts

### Stack Fundamentals
- LIFO (Last In, First Out) data structure
- Push: Add element to top
- Pop: Remove element from top
- Peek: View top element without removing

### Implementations
- **Java Stack**: Built-in class
- **Linked List Stack**: Dynamic size
- **Array Stack**: Fixed capacity, better performance

### Applications
- Function call stack
- Expression evaluation
- Undo/redo operations
- Browser history navigation

### Time Complexity
- Push: O(1)
- Pop: O(1)
- Peek: O(1)
- Search: O(n)

### Common Uses
- Palindrome checking
- Bracket balancing
- String reversal
- Postfix expression evaluation
