# Queues Example

## File Overview
Demonstrates queue implementations (built-in, custom linked list, circular), priority queues, deques, and real-world applications like task scheduling and print queue simulation.

## Key Concepts

### Queue Fundamentals
- FIFO (First In, First Out) data structure
- Enqueue: Add element to rear
- Dequeue: Remove element from front
- Peek: View front element without removing

### Implementations
- **Java Queue**: Built-in interface
- **Linked List Queue**: Dynamic size
- **Circular Queue**: Fixed capacity, efficient space usage
- **Priority Queue**: Elements processed by priority
- **Deque**: Double-ended queue

### Applications
- Task scheduling
- Print queue management
- BFS algorithms
- Message queuing systems

### Time Complexity
- Enqueue: O(1)
- Dequeue: O(1)
- Peek: O(1)
- Search: O(n)

### Common Uses
- Job scheduling
- Print spooling
- Message passing
- Resource management
