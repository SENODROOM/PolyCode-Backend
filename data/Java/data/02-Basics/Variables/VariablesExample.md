# Variables Example

## File Overview
This file demonstrates different types of variables in Java programming.

## Key Concepts Explained

### 1. Primitive Variables
- **int age = 25**: Integer variable for whole numbers
- **double price = 99.99**: Double precision floating-point number
- **char grade = 'A'**: Single character
- **boolean isActive = true**: Boolean value (true/false)

### 2. Reference Variables
- **String name = "John Doe"**: String object reference
- **int[] numbers = {1, 2, 3, 4, 5}**: Array reference

### 3. Constants
- **final double PI = 3.14159**: Final variable that cannot be changed once assigned

### 4. Variable Types by Scope
- **Static Variable**: Belongs to the class, shared among all instances
- **Instance Variable**: Belongs to an object instance
- **Local Variable**: Declared within a method, only accessible within that method

## Variable Declaration Rules
1. Variable names must start with a letter, underscore, or dollar sign
2. Cannot start with a number
3. Cannot use Java keywords
4. Should be descriptive and follow camelCase convention

## Best Practices
- Use meaningful variable names
- Initialize variables before use
- Use final for constants
- Follow naming conventions
