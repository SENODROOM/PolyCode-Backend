## Books

### Beginner
- **"Quantum Computing: An Applied Approach"** — Jack Hidary  
  Great balance of theory and code (includes Q#)

- **"Programming Quantum Computers"** — Johnston, Harrigan, Gimeno-Segovia  
  Hands-on with circuits and code

### Intermediate
- **"Quantum Computation and Quantum Information"** — Nielsen & Chuang  
  The definitive textbook ("Mike and Ike") — dense but comprehensive

- **"An Introduction to Quantum Computing"** — Kaye, Laflamme, Mosca  
  More accessible than Nielsen & Chuang

### Advanced
- **"Quantum Error Correction"** — Lidar & Brun  
  Comprehensive error correction reference

- **"The Theory of Quantum Information"** — John Watrous  
  Free online, graduate-level theory

## Glossary

| Term | Definition |
|------|-----------|
| **Qubit** | Quantum bit; can be in superposition of 0 and 1 |
| **Superposition** | Quantum state that is a combination of basis states |
| **Entanglement** | Correlated quantum states of multiple qubits |
| **Decoherence** | Loss of quantum properties due to environment |
| **Unitary** | Reversible quantum operation (gate) |
| **Measurement** | Extracting classical information, collapses quantum state |
| **Ancilla** | Helper qubit used temporarily in a computation |
| **Oracle** | Black-box quantum operation encoding a classical function |
| **NISQ** | Noisy Intermediate-Scale Quantum (current hardware era) |
| **Fault-tolerant QC** | Error-corrected quantum computing (future) |
| **Circuit depth** | Number of sequential gate layers |
| **T-count** | Number of T-gates; key resource metric |
| **Logical qubit** | Error-protected virtual qubit |
| **Physical qubit** | Actual hardware qubit |
| **Code distance** | Error correction strength metric |
| **Threshold** | Error rate below which more qubits = better |
| **Variational circuit** | Parameterized quantum circuit for hybrid algorithms |
| **Hamiltonian** | Operator representing energy of a quantum system |
| **Eigensolver** | Algorithm to find eigenvalues/vectors of a matrix |
| **QFT** | Quantum Fourier Transform |
| **QPE** | Quantum Phase Estimation |
| **QAOA** | Quantum Approximate Optimization Algorithm |
| **VQE** | Variational Quantum Eigensolver |
