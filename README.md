## Prerequisites
- Basic programming knowledge (any language)
- Terminal / command line comfort

## Setup
```bash
# Install Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Verify
rustc --version
cargo --version

# Create a new project
cargo new my_project
cd my_project
cargo run
```
